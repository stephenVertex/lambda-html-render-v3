Google Doc Style Test

This is a test of the Google Doc window decorator style, which renders markdown natively like a Google document.

Features

  • No title bar or window decorations
  • Rendered markdown (not just plain text)
  • 8.5x11 paper-like aspect ratio
  • Clean, minimal interface

Code Example

def hello_world():
    print("Hello from Google Doc Style!")
    return True

Bold text, italic text, and hyperlinks all work as expected.
